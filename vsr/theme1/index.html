ВСР Тема 1. Сборка проекта с помощью бандлера Webpack

  1. Добавление зависимостей webpack и webpack-cli 
    
        yarn add -D webpack webpack-cli


  2. Добавление библиотеки moment 
    
        yarn add moment


  3. Создание файла index.js
  4. Сборка проекта 
    
        yarn webpack . -o ./bundle.js
